Titulo


Para entrar al Control de Maestros:

Acceder a: http://localhost:8080/Maestro/index

Mejoras

  • En este Parcial 3 se mejoró la funcionalidad del programa del Parcial 2 añadiendo una base de datos en H2 en lugar de usar una lista.
  • Se agregó un nuevo botón donde podemos ver una lista de maestros con botones individuales por cada maestro para editarlo o eliminarlo.
  • Se agregó un servicio REST para poder consumir las funcionalidades CRUD a través de una URL.

REST

* Punto 4 de este parcial.

Funcion URL
Mostrar todos los maestros http://localhost:8080/restMaestro
Mostrar un maestro especifico http://localhost:8080/restMaestro/1
Agregar un maestro http://localhost:8080/restMaestro/agregar/100/Paco/24/15500.0
Editar un maestro http://localhost:8080/restMaestro/editar/100/Pancho/23/20500.0
Eliminar un maestro especifico http://localhost:8080/restMaestro/eliminar/100
